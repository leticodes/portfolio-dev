## 💼 Repositório de Portfólio

Este repositório foi criado para armazenar projetos desenvolvidos durante meus estudos em desenvolvimento front-end.

## 🎯 Objetivo

Praticar conceitos de desenvolvimento, aprimorar a lógica e evoluir minhas habilidades através da criação de projetos práticos e interfaces.

## 🚀 Tecnologias utilizadas

* HTML

* CSS

* JavaScript

* React

* Git e GitHub

## 📂 Estrutura

Os projetos estão organizados em pastas, onde cada uma representa um projeto diferente, facilitando a navegação e visualização da evolução.

## 💡 Observação

Este repositório faz parte do meu processo de aprendizado, portanto os projetos podem ser atualizados, melhorados ou refatorados ao longo do tempo.
